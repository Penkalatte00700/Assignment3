База даних для керування замовленнями в інтернет-магазині на PostgreSQL.
Є таблиці для клієнтів, товарів, замовлень і логів. Реалізовані функція для підрахунку суми замовлення(calculate_order_total) процедури для створення замовлень і додавання товарів. 
Також створив два тригери: один оновлює суму замовлення автоматично, а інший водночас логує нові замовлення.
Все знаходиться в solution.sql.
